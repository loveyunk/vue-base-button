# vue-base-button

> Vue Base Button Component

## Install

```bash
npm install @loveyunk/vue-base-button --save
```

## Usage

```js
<template>
  <vue-base-button>Button</vue-base-button>
</template>

<script>
  import VueBaseButton from '@loveyunk/vue-base-button';
  export default {
    components: { VueBaseButton }
  }
</script>
```
